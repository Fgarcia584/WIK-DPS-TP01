# DevOps WIK-DPS-TP01

## Installation des dépendances

Run `npm install` 

```
npm install
```

## Démarrer le serveur web

```
node ./build/index
```

Modifier la variable d'environnement `PING_LISTEN_PORT` permet de modifier le port d'écoute du serveur :

```
PING_LISTEN_PORT=3000 node ./build/index
```
